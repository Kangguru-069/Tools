Microsploit Tools

Microsploit adalah sebuah tools sederhana yang digunakan untu membuat eksploitasi backdoor pada Microsoft Office di windows atau mac, Open Office di linux, Macro Attack, Buffer Overflow di word dengan menggunakan modul metasploit packet.

1. Install Microsploit
   $ git clone https://github.com/Screetsec/Microsploit.git
   
2. Memberikan Izin Akses
   $ chmod +x Microsploit

3. Jalankan Program
   $ ./Microsploit
